# hifipi
Projektdokumentation für eine Musikbox mit einem Raspberrypi, Hifiberry AMP. Einbau zusammen mit Lautsprechern und Steuerelementen (Drehencoder für Lautstärke / Auswahl, Buttons für Steuerung)
